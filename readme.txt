== Description ==

Plugin to create projects of transformation, photo shoots, shopping advice, workshops and page output archives (data projects) in a design. When changing themes the design of the archive pages is the same, except for caps, basement and environment of the content (comes from theme). All written projects are in the detailed description take the design theme.
